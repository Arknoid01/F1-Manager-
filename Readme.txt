🏎️ F1 Manager

Un jeu de management F1 jouable directement dans le navigateur, développé en HTML, CSS et JavaScript.

Le projet propose une expérience complète de gestion d’équipe : R&D, staff, entraînement, médias, stratégie de course, objectifs de la direction, carrière sur plusieurs saisons et bien plus.
